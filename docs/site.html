GitHub Repository Initializer

Streamlining your project setup on GitHub

Overview

This application facilitates the process of turning a local project into a Git repository and pushing it to GitHub, automating initial setup tasks like repository creation, file addition, and initial commit.

How to Use

  1. Ensure Java and Git are installed on your machine.
  2. Download and run the application.
  3. Enter your project details in the provided GUI.
  4. Click "Initialize and Push" to create your repository on GitHub.
